A peaceful web application that allows users to listen to the Holy Quran while enjoying beautiful animated video backgrounds.

The project is built entirely with **HTML, CSS, and Vanilla JavaScript**, and serves as one of my first personal JavaScript projects after completing problem-solving fundamentals.

---

## ✨ Features

* 🎧 Upload and play local Quran audio files.
* 🎬 Change the background video from your device.
* ▶️ Custom Play / Pause controls.
* 🔊 Mute / Unmute controls.
* 📄 Display the selected Surah file name.
* ✨ Smooth UI animations.
* 🪟 Glassmorphism interface.
* 📱 Responsive layout.
* ✅ File validation.
* 🚫 Prevent multiple audio players from being created.
